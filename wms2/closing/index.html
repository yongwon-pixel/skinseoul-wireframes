동작 규칙(화면 밖): 마감 시작(수동 카운팅 입력) 전에는 스캔창 비활성. 마감 시작 후에는 페이지 아무 곳 클릭 시 스캔창 자동 포커스(전체 선택)·연속 스캔, 처리 결과는 새로고침 없이 우상단 토스트 — 화면 문구 없이 동작으로만 구현. 스캔한 송장의 오더 상태를 즉시 조회 → Prepare Shipment면 정상(초록·무음), 그 외 상태(Processing 등)거나 오늘 이미 스캔된 송장(중복)이면 경고(빨강·음성). 마감 확정 조건 = 정상 스캔 수가 수동 입력한 오늘 출고 대상과 정확히 일치 + 경고 0건(초과 스캔도 불일치). Processing 건 처리(M1)의 Outbound는 실제 오더 상태 변경으로 전 화면에 반영. 매칭 오더가 없는 송장은 "없는 주문" 경고(상태 2b) — 마감 카운트 미포함. 마감 진행 상태는 서버 저장 — 스캔 도중 다른 페이지로 이탈했다 돌아와도, 새로고침해도 카운팅·스캔 리스트·경고가 그대로 유지되며 마감 확정 또는 명시적 재시작 전까지 클리어되지 않음. 연속 스캔: 스캐너가 붙여주는 자동 Enter가 제출을 대신하며(스캔 버튼 클릭 불필요), 판정 직후 커서는 스캔창에 전체 선택 상태로 복귀 — 다음 스캔이 이전 값을 덮어써 몇 건이든 끊김 없이 연속 스캔.
